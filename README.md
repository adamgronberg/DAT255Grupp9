DAT255Grupp9
============






ALPHA RELEASE 0.1 

features:
*Move left and right with arrrows
*You die when a enemy collides with you
*Enemys spawn forever even if you die
*Enemies die when you shoot them
*To restart game close(press back button on device) and open again

How to launch:
1. Import the projects to eclipse
*swipe-race-turtorial
*swipe-race-turtorial-android
*swipe-race-turtorial-desktop

2. Check the Android 4.3 target in properties->Android
3. Select either the desktop project or android project and launch it
	*Select the class Main.java in desktop version
